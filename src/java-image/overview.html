This is a small Java library for writing GIF images.

Following classes are essential for finishing the implementation:

  • {@link cz.alisma.alej.prog.image.RasterImage} that represents a raster image.
  • {@link cz.alisma.alej.prog.image.Color} representing RGB color and
  • {@link cz.alisma.alej.prog.image.gif.GifWriter} contains almost finished implementation of GIF encoder.
